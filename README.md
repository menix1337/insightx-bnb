# InsightX – BNB Chain Deployment

InsightX is integrating with BNB Chain (BSC) to provide on-chain analytics, wallet intelligence, and token behavior insights for the BNB ecosystem.

This repository contains configuration and deployment references for running InsightX components on BNB Chain.

## Deployment Targets
- **BNB Smart Chain (BSC mainnet)** – chainId: 56  
- **BNB Testnet** – chainId: 97
